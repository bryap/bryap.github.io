# Brya K. Patterson Portfolio

Welcome to my professional portfolio. This site showcases my work in Data Science, Data Engineering, Scrum Master leadership, and specialized research. I am a data professional based in Portland, Oregon, with a passion for turning complex information into actionable insights.

## 🚀 Overview

This repository contains the source code for my personal portfolio website, hosted at [bryap.github.io](https://bryap.github.io). The site is built with a focus on clean organization, responsive design, and an interactive project filtering system.

## 🛠️ Technical Stack

- **Languages:** Python, SQL, R, JavaScript, HTML5, CSS3
- **Data Science & ML:** TensorFlow, Pandas, DuckDB, Predictive Modeling
- **Frameworks & Tools:** Scrum/Agile, Jira Automation, Git
- **Design:** Flexbox-based responsive layout, custom CSS/JS filtering logic

## 📂 Project Highlights

- **Scrum for Data Analytics:** A case study on adapting Agile frameworks for non-product data teams.
- **NBA Draft Combine Predictions:** A Neural Network model evaluating the ROI of "pure athleticism" in the NBA.
- **FishkeepingDB:** A 16-table normalized relational database for complex biological and retail inventory.
- **Cybersecurity Log Reconstruction:** Forensic investigation into Business Email Compromise using Python and SQL.
- **My Color Picker:** A secure Chrome Browser Extension utilizing the native EyeDropper API.

## 🔧 Maintenance & Organization

The codebase has been refactored for modern standards:
- **Separation of Concerns:** Embedded styles and scripts have been moved to external files (`assets/css/` and `assets/js/`).
- **Optimization:** Removed redundant template placeholders and optimized for fast loading.
- **Interactive Filtering:** Custom JavaScript implementation for dynamic project discovery based on domains and expertise.
